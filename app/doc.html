Lang Wiki API

Methods

Search by language

Endpoint: http://localhost:5000/search_lang

Request type: Get

Request body: "lang" field name 

  http://localhost:5000/search_lang?lang=French

Response: JSON with parse details, e.g. 

    [
  {
    "Branch": "Romance",
    "Countries": 51,
    "Family": "Indo-European",
    "ISO-Code": "fr",
    "Name": "French",
    "Scripts": [
      "Latin"
    ],
    "Speakers": 312
  }
]

Request type: Get

Request body: "family" field name 

  http://localhost:5000/search_family?family=Dravidian
