Lang Wiki API

Methods

See all languages

Endpoint: http://localhost:5000/languages

Request type: Get

Request body: N/A

  http://localhost:5000/languages

Response: JSON with several language entries

Search by language

Endpoint: http://localhost:5000/search_lang

Request type: Get

Request body: "lang" field name 

  http://localhost:5000/search_lang?lang=French

Response: JSON with that language details, e.g. 

   [
  {
    "French": {
      "Branch": "Romance",
      "Countries": 51,
      "Family": "Indo-European",
      "ISO-Code": "fr",
      "Scripts": [
        "Latin"
      ],
      "Speakers": 312
    }
  }
]

Request body: "family" field name
